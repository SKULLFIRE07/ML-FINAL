ML Pipeline
1

Canvas Capture

280x280 HTML canvas drawing is captured and sent as base64 PNG to the API.

2

Preprocessing

Image is inverted, bounding-box cropped, resized to 20x20 and centered in 28x28 to match MNIST format.

3

Deskewing

Moment-based affine transformation corrects digit slant, reducing error rate by ~45%.

4

PCA Reduction

784 features reduced to 50 principal components retaining 87.1% variance. Speeds up inference 15x.

5

KNN Classification

Distance-weighted K=3 nearest neighbors voting with Minkowski distance predicts the digit.
